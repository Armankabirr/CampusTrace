# Campus Images for Authentication Pages

This directory contains background images for the login and signup pages.

## Instructions to Add Images

1. Save the two campus images you provided as:
   - `campus-1.jpg` - The image with the large university buildings and green lawn
   - `campus-2.jpg` - The image with the pink flowers and "UIO" sign building

2. Place them in this directory: `public/images/`

The authentication page will automatically use these images as backgrounds for the login and signup forms.

## Image Requirements

- Format: JPG, PNG, or WebP
- Minimum dimensions: 1920x1080px (recommended)
- File size: Keep under 500KB for optimal performance

Once you add the images:
- The login form will use campus-1.jpg as the background
- The signup form will use campus-2.jpg as the background
- If images are not found, the page will display a fallback gradient background

## Fallback Background

If the images are not yet added, the page will display a beautiful orange-to-red gradient background, ensuring the authentication pages are always functional.
